# No Delay 🚀
### Delay Ends Here

## What is No Delay?
No Delay is a psychology-based anti-procrastination web app that forces users to START tasks and tracks WHY they stop.

## Features
- ✅ Add up to 3 tasks (cognitive load principle)
- 🔒 Focus Mode — locks UI during task
- ⏱️ 25-minute Pomodoro Timer
- 🧠 Psychological Delay System
- 📊 Behavior Tracking (delay count + reasons)
- 🔥 Day Streak System
- 💾 Local Storage — data persists
- 🎨 Clean Dark UI

## Tech Stack
- React (useState, useEffect)
- CSS3 (Dark theme, animations)
- LocalStorage API

## How to Run
```bash
npm install
npm start
```
